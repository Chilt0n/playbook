P21: Wipe device

Issue Explanation

A user may want to start fresh on a device and generate a new private key. In some rare cases wiping a device and restoring would solve functionality issues in the platform, but this step should only be used as last resort for functionality issues.

Follow these steps:

  1. P21 - T1: Confirm you have your 12-word recovery sentence.
  2. P21 - T2: Start from beta.shapeshift.com
  3. P21 - T3: Connect your device (Follow P1-T1 for detailed instructions).
  4. P21 - T4: Select ‘Settings’
  5. P21 - T5: Select ‘Wipe’
  6. P21 - T6: Confirm that you want to wipe your device.
  7. P21 - T7: Hold down the button on your KeepKey to confirm and wipe the device.
  8. P21 - T8: To restore Begin initialization process see P4 [link]
  9. P21 - T9: If unable to assist customer with the above, escalate to: change Assignee > Escalations and leave internal note with details on issue.

Post-troubleshooting Steps

  • Step 1: Tag support ticket with CODE [ code = Issue # + Step you solved the problem with in the troubleshooting list, example: P1T2 (platform issue 1, troubleshooting step 2). If you can not solve with any of the above steps, tag as PXXNONE.
