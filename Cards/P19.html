P19: Switch Wallets

Issue Explanation

Currently, the ShapeShift platform only supports one wallet at a time. When you switch wallets, your first wallet will be ejected and forgotten from the platform until you reconnect. This will NOT affect the data or assets on that wallet.

Follow these steps:

  1. P19 - T1: Ask if user is if the wallet they are switching to connecting a new wallet that has not been initialized or a wallet that’s already been set up (Do you already have the recovery sentence? If no, this is a new wallet, if yes, it’s already been set up).
  2. P19 - T2: P19-T2 On the top right corner, click the drop-down arrow and select “Switch Wallet” to confirm that you’d like to switch the wallet you’re using on the Platform.
  3. P19 - T3: Check the “I understand” box to confirm the message above.
  4. P19 - T4: Select which crypto wallet you’d like to connect:
  5. P19 - T5: Pair your new wallet by accepting the Terms of Service and Privacy Policy by checking the box. Then click the “Pair KeepKey” button to continue.
  6. P19 - T6: In the pop up window on the left, highlight the wallet you would like to pair and then click "Connect". If device is not detected, have user refresh and try again, if it still fails to connect move to P1 [link to P1]
  7. P19 - T7: If new wallet begin initialization [link to this section] (Reference P19-T1)
  8. P19 - T8: If device is already set up, confirm your PIN to enter your device onto the ShapeShift Platform!
  9. P19 - T9: Once PIN has been entered successfully your device will be connected to the platform and you can view balances. Done!.
  10. P19 - T10: If unable to assist customer with the above, escalate to: change Assignee > Escalations and leave internal note.
